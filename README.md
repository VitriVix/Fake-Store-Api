# Fake Store API - Frontend

## Instrucciones de Instalación

Sigue los siguientes pasos para correr el proyecto localmente:

1. **Clona el repositorio** en tu máquina local:
   ```en tu terminal:
   git clone https://github.com/VitriVix/Fake-Store-Api.git
   
2. Navega al directorio del proyecto:
cd tu-repositorio

3. Instala las dependencias necesarias utilizando npm:
npm install

4. Corre el proyecto en modo de desarrollo con Vite:
npm run dev

5. Abre el navegador y visita la siguiente URL:
http://localhost:5173


Resumen de Tecnologías Utilizadas

React.js: Usé React porque hace que crear interfaces sea más fácil y rápido, además de que es muy práctico para hacer componentes reutilizables y mantener el código organizado.

Vite: Elegí Vite porque es súper rápido al cargar el proyecto y permite ver los cambios al instante mientras desarrollo.

CSS Grid con Media Queries: Utilicé CSS Grid para hacer una grilla que se adapta a cualquier tamaño de pantalla, así que la interfaz se ve bien en móviles y pantallas grandes.

JavaScript (ES6+): JavaScript es esencial para que la app funcione. Usé ES6+ porque tiene mejoras que hacen que el código sea más simple y limpio.

npm: Lo usé para manejar todas las librerías y dependencias del proyecto, manteniéndolo todo organizado y fácil de manejar.
